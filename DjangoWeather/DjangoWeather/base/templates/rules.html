Règles

Bienvenue sur la Casino de la Météo !
Le but du jeu est simple. C'est un jeu de hasard où trois roues tournent simultanément avant d'afficher un temps sur lesquels vous pouvez parier en amont.
En fonction des combos que vous créer, vous n'obtenez pas les mêmes récompenses, c'est à vous de réfléchir sur ce que vous souhaitez gagner afin d'avoir un maximum de moula à la fin ! Vous choisissez un temps, puis vous misez la somme que vous voulez, plus vous miser une grande somme, plus les gains seront élevés ! Mais attention, certaines combinaisons peuvent être des malus...

Voici les paramètres du jeu :

Soleil (5%)

1 Soleil = mise x 5 + 1 free lancer (5%)
2 Soleil = mise x 10 + 2 free lancer (0,25%)
3 Soleil = mise x 15 + 3 free lancer (0,0001%)

Pluie (30%)

1 pluies = rien (30%)
2 Pluies = 3 free lancers (9%)
3 Pluies = 5 free lancers (2,7%)

Vent (15%)

1 vent = rien (15%)
2 Vent = gain x 1,5 (2,25%)
3 Vent = gain x 3 (0,3375%)

Nuages (30%)

1 nuage = rien (30%)
2 nuages (9%)
3 Nuages = Deviner une température (tu trouve pas il se passe rien sinon gain x 2) (2,7%)

Événements perturbateurs (20%) de chances de tomber sur un :

Orages (10%)

1 Orage = rien (10%)
2/3 Orages = Perdu (1%)

Chances (10%)

1 chance = rien (10%)
2 chances = rien (1%)
3 Chances = gain x multiplicateur aléatoire (modifié) (0,1%)
